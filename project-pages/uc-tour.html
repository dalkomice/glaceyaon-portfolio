University of the Cordilleras Virtual Tour Mockup
A virtual campus tour to be released in the near future
Project Details / Background
This project introduced me to the use of Unity which utilizes the C++ language. 4 of us worked on the design of the logo and buttons then the 2 of us integrated those graphics to the interface design. I was also able to use Unity when we were linking one area to the other. This project was finished in 1 month.
The tour uses the same audio in the university's promotional video and the same script as well. Graphics were created in Adobe Photoshop and Adobe Illustration which is used in Figma then transferred the work to Adobe XD for us to collaborate in the cloud.
